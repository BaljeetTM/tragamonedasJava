Tragamonedas en Java
-
Proyecto realizado en Java sobre un tragamonedas, donde
un usuario puede realizar apuestas y ganar premios
dependiendo de la combinación de las ruedas.

Se entregó un proyecto inicial en el cual se solicitó
mover la lógica principal desde la clase Box a TragamonedasImpl.

INSTRUCCIONES DE EJECUCIÓN
-
- Clonar o descargar el repositorio de GitHub
- Abrir el proyecto
- Dirigirse a la clase TheMain en la carpeta src
- Ejecutar el programa
